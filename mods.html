VoiceDash

This mod allows you to play GD with your voice. All you need is Mega Hack, where you'll move the main DLL to the extensions folder, move the portaudio DLL into the game's directory, then start game and boom. It has an adjustable audio threshold to help with sensitivity and all that which is pretty cool I think. It technically works with other games too but I wouldn't try it if it has an anticheat.
