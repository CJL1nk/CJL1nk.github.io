Fake Moderator

A pretty simple Geode mod that gives you a Mod badge, a "working" REQ button, and a "working" mod suggest stars button. Currently there's only a regular mod option but in the future I'm gonna add an option for elder mod aswell and possibly comment coloring.

VoiceInput

This program basically just detects voice input and holds down left click while it's detected. It has an adjustable sensitivity stored in the holy .john file and yeah that's really it. It's really fun to try and play games with but I wouldn't try to use it on games with an anticheat cuz like who knows how that might go.
